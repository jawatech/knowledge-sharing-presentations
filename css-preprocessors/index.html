What is a CSS preprocessor?

CSS pre-processor is a tool that compiles syntaxically enhanced style language to plain CSS

What useful features is CSS missing?

  1. Variables
  2. Nested rules
  3. Mixins
  4. Modules

The evolution of preprocessing

2007 – SCSS (Ruby)

2009 – LESS (JS)

2010 – Stylus (JS)

2013 – PostCSS (JS)
