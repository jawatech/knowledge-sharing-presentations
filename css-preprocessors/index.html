What is a CSS preprocessor?

CSS pre-processor is a tool that compiles syntaxically enhanced style language to plain CSS

Why do we need preprocessing?

  1. The CSS standard is evolving slowly.
  2. We want to use new features today.
  3. We want to use better syntax.

What useful features is CSS missing?

  1. Variables
  2. Nested rules
  3. Mixins
  4. Modules
  5. Calculations

The evolution of preprocessing

2007 – SASS (SCSS) (Ruby)

2009 – LESS (JS)

2010 – Stylus (JS)

2013 – PostCSS (JS)

Benefits of using LESS:

  • Less code than with CSS
  • Language features
  • LESS is pure JavaScript (~5x faster than SCSS)
